Harvest For Good API

Welcome to the Harvest For Good API. This page provides basic information about available endpoints.

This is a backend API server. For the full application, please use the frontend application running on http://localhost:3000.

Authentication Endpoints

POST /api/token/

Obtain a JSON Web Token by providing your credentials.

Example request body:

{
  "email": "user@example.com",
  "password": "yourpassword"
}
POST /api/token/refresh/

Refresh your token before it expires.

Example request body:

{
  "refresh": "your_refresh_token"
}
POST /api/users/register/

Register a new user account.

User Endpoints

GET /api/users/me/

Get information about the currently authenticated user.

Forum Endpoints

GET /api/forum/posts/

List all forum posts. This endpoint is publicly accessible.

POST /api/forum/posts/

Create a new forum post. Authentication required.

GET /api/forum/posts/{id}/

View a specific forum post. This endpoint is publicly accessible.

Academic Endpoints

GET /api/academic/

Access academic resources. Authentication required.

For more detailed API documentation or assistance, please contact the development team.
